# Автоматизация формы sixmo.ru

Этот skill автоматически проходит многошаговую тестовую форму на https://sixmo.ru/

## Что делает

1. Открывает сайт sixmo.ru в браузере (Playwright + Chromium)
2. Нажимает "Начать задание"
3. Заполняет Этап 1 (2 текстовых поля + 1 select — порядок случайный)
4. Заполняет Этап 2 (1 текстовое поле + 1 select + загрузка файла — порядок случайный)
5. Отправляет форму и получает идентификатор результата

## Как использовать

Запусти скрипт командой:

```bash
node sixmo-form.js
```

### С пользовательскими данными

Можно передать JSON с данными для заполнения через `--data`:

```bash
node sixmo-form.js --data='{"step1_text1":"Букля","step1_select":"Хогвартс","step1_text2":"Гриффиндор","step2_text":"Платформа 9 3/4","step2_select":"Снитч","file_path":"./my_file.txt"}'
```

### Параметры

| Параметр | Описание | По умолчанию |
|----------|----------|-------------|
| `step1_text1` | Ответ на 1-й текстовый вопрос этапа 1 | Из placeholder |
| `step1_select` | Текст варианта select этапа 1 | Первый вариант |
| `step1_text2` | Ответ на 2-й текстовый вопрос этапа 1 | Из placeholder |
| `step2_text` | Ответ на текстовый вопрос этапа 2 | Из placeholder |
| `step2_select` | Текст варианта select этапа 2 | Первый вариант |
| `file_path` | Путь к файлу для загрузки (.txt/.md/.json до 50КБ) | Временный файл с текстом "Expelliarmus" |
| `headless` | Запуск без UI | `false` (сайт блокирует headless) |

### Результат

Скрипт выводит JSON с результатом:

```json
{
  "success": true,
  "identifier": "24E0AF1831CB",
  "flowId": "abc123...",
  "url": "https://sixmo.ru/#/flow/.../result",
  "screenshot": "sixmo_result.png"
}
```

## Важные особенности

- Поля на каждом этапе появляются в **случайном порядке** — скрипт определяет тип поля (text/select/file) динамически
- DOM-структура **плавающая** — ID и классы элементов меняются при каждом запуске
- Между этапами есть **задержки загрузки** — скрипт ожидает появления контента
- Сайт **блокирует headless**-браузеры — поэтому по умолчанию запуск с видимым окном
